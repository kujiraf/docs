AWS CLI¶

Install¶

asdfでバージョン管理を行う。

# add plugin
asdf plugin add awscli
# show plugin list
asdf plugin list
# install
asdf install awscli latest
# show install list
asdf list
# set primary version
asdf global awscli 2.8.6

Profileの変更¶

# change profile
export AWS_PROFILE=[~/.aws/credentials上のprofile]
# show identity
aws sts get-caller-identity
